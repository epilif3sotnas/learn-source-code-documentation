## About learn-source-documentation

This repository was built with the goal to learn about source code documentation and its HTML generation in Nim, Java, and Go.

&nbsp;


## Requirements

#### Nim    => Package.nimble

#### Java   => pom.xml

#### Go     => go.mod

&nbsp;
